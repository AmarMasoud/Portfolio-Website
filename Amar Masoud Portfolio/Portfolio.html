    My Projects

    In this section, you’ll find some of the projects I have worked on to build and showcase my skills. Each project reflects my experience with HTML, CSS, and JavaScript, and highlights my passion for creating clean, responsive, and user-friendly websites. I am proud of the progress I’ve made so far and excited to keep learning and adding even more projects in the future.

    Apple Game

    This project is a fun and interactive game where the player controls a basket using the keyboard to catch falling apples. Each time an apple is successfully caught, the player’s score increases by one. The game is simple, engaging, and highlights my skills in handling user input, dynamic elements, and score tracking.

    Calculator

    This project is a fully functional calculator that performs basic arithmetic operations just like a real calculator. I focused on making the calculations accurate and the user experience smooth. To enhance its appearance, I used CSS to design a clean, modern, and visually appealing interface.

    Class Manipulator

    This project is a class manipulator tool that allows users to easily add and remove words. It features two input fields: one for adding words and another for removing them. When a user enters a word in the add input, the word is added to the class list, and when a word is entered in the remove input, it gets removed from the class. This project demonstrates my understanding of DOM manipulation and event handling in JavaScript.

    Clock

    This project is a complete clocks website that brings together everything related to timekeeping. It includes a wall clock, a stopwatch, a timer, and a digital clock, all in one place. I designed and built this website to showcase my ability to manage different functionalities, user interactions.

    E-commerce

    This project is an E-commerce website focused on showcasing Jordan shoes. I designed it to be visually appealing, easy to navigate, and user-friendly. The site highlights my skills in creating clean layouts, attractive product displays, and smooth user experiences, making it a strong example of my front-end development abilities.

    Interactive Resume

    This project is an interactive resume that presents information in a clean and engaging way. While it's simple in design, it effectively highlights key details with smooth interactions and a modern layout. It shows my ability to create professional, user-friendly interfaces.

    Quiz App

    This project is a quiz application that features three levels of difficulty: easy, medium, and hard. Each question comes with four possible answers, and the user selects one. If the choice is correct, the score increases by one. At the end of the quiz, the user's total score is displayed.

    To-Do List

    This project is a To-Do List application that allows users to add tasks through an input field. Each task is saved in the browser’s local storage, ensuring that the tasks remain even after the website is closed and reopened. This project demonstrates my ability to work with local storage, manage data persistence, and create simple yet effective user experiences.

    Weather App

    My weather app project is a great way to provide users with real-time weather updates. By allowing the users to input a country or government, the app retrieves and displays not only the current day's weather but also forecasts for the next five days. This feature ensures users can plan ahead and stay informed. The app's practical design and functionality make it a standout project.
